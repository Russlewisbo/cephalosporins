## Notes on Image Sourcing

- **DO NOT** search for copyrighted images online
- Chemical structures can be drawn using ChemDraw or similar software
- Mechanism diagrams can be created de novo
- Consider using Creative Commons licensed images from Wikimedia Commons for:
  - Bacterial cell wall structure
  - General β-lactam mechanism diagrams

## File Naming Convention

Use descriptive names:
- `fig01_cephem_nucleus.png`
- `fig02_first_gen_structures.png`
- `mechanism_pbp_binding.png`
- etc.

## Recommended Image Specifications

- Format: PNG (for diagrams) or SVG (for scalable graphics)
- Resolution: Minimum 300 DPI for print, 150 DPI for web
- Background: Transparent or white
- Size: Appropriate for slide presentation (typically 1920x1080 max)
